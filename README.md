# Native Heart Widget

Android home-screen widget prototype for a small paired-heart experience.

The Android project lives in:

```text
native-heart-widget
```

See the project README:

```text
native-heart-widget/README.md
```

## Build

```powershell
cd native-heart-widget
.\gradlew.bat assembleDebug
```

The debug APK is produced at:

```text
native-heart-widget/app/build/outputs/apk/debug/app-debug.apk
```

## Notes

Local SDK/JDK/toolchain folders are intentionally ignored:

- `.tools`
- `android-sdk`
- `.gradle-cache`
